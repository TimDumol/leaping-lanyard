Angular Momentum Lesson 1

This simple app posts messages on a public board anonymously.

Explanation:

GET gets a message from the server given an id. Ignores the message field.

POST creates a new message on the server. Ignores the id field.

PUT updates a message on the server given the id and the new message.

DELETE deletes a message on the server given the id. Ignores the message field.
